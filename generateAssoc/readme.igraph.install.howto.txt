To install iGraph on linux

- install required packages
	install build-essential libxml2-dev libglpk-dev libgmp3-dev libblas-dev liblapack-dev libarpack2-dev python-dev

- Download igraph from    http://igraph.org/python/#downloads     (source code). I managed to install python-igraph-0.7.0 on ubuntu 14.01 with Python 2.7.6 
- Extract
- go to the directory from the terminal
	python setup.py build
	sudo python setup.py install
